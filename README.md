# Kaelor Systems

**Building games, creative tools, and intelligent systems.**

Kaelor Systems is an independent software studio focused on creating games, creative software, AI-powered tools, and experimental systems.

Our main project is **StageVerse**, a rhythm game and creative platform focused on music, performance, and community-made content.

## Projects

- **StageVerse** — Rhythm game and creative music platform.
- **StageVerse Portal** — Community, profiles, uploads, charts and creator tools.
- **StageVerse Tools** — Audio analysis, chart generation, importers and development utilities.
- **Kaelor Media Vault** — Media organization and research tools.

## Vision

We believe in building tools and experiences that combine creativity, technology, music, and intelligent systems.

Kaelor Systems is still in its early stage, but every project is being built with long-term vision, technical discipline, and creative ambition.
